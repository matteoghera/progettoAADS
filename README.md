# Progetto di *Analysis of Algorithms and Data Structures*

Autori: Luisa Collodi luisa.collodi@stud.unifi.it, Matteo Ghera matteo.ghera@stud.unifi.it, Federico Fiorentino federico.fiorentino@stud.unifi.it

L'obiettivo di questo progetto è l'analisi dei *2-Motzkin paths*, attraverso la verifica di alcuni risultati che li riguardano, sulla base dell'articolo: [*"The Fine numbers refined"*] [article] di Cheon, Gu-Lee e Shapiro.
